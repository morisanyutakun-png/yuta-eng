## <span id="h2-basics">LaTeX 数式 物理｜まず押さえる基本記法10選</span>

**LaTeX 数式 物理** で最も登場頻度が高い10項目を最初に置きます。これだけで定期テスト〜共通テスト相当の問題は8割書けます。

| 用途 | LaTeX ソース | レンダリング |
| --- | --- | --- |
| インライン数式 | `$v=v_0+at$` | $v=v_0+at$ |
| ブロック数式 | `$$E=mc^2$$` | $E=mc^2$ |
| 添字（下） | `v_0`、`v_{0}` | $v_0,\ v_{0}$ |
| 添字（上） | `v^2`、`v^{2x}` | $v^2,\ v^{2x}$ |
| 分数（小） | `\tfrac{1}{2}` | $\tfrac{1}{2}$ |
| 分数（大） | `\dfrac{1}{2}` | $\dfrac{1}{2}$ |
| 平方根 | `\sqrt{2gh}` | $\sqrt{2gh}$ |
| n乗根 | `\sqrt[3]{V}` | $\sqrt[3]{V}$ |
| 不等号（≤, ≥） | `\leq, \geq` | $\leq,\ \geq$ |
| 概ね等しい | `\approx, \sim` | $\approx,\ \sim$ |

<Callout variant="tip" title="\\tfrac と \\dfrac の使い分け">
**$\tfrac{1}{2}$（tfrac）** は本文中の小さい分数、**$\dfrac{1}{2}$（dfrac）** はブロック式での大きな分数。物理の式で「$\tfrac{1}{2}m v^2$」のように書くと組版が美しくなります。
</Callout>

## <span id="h2-vectors">LaTeX ベクトル 表記｜\\vec, \\bm, 内積・外積</span>

**LaTeX ベクトル 表記** には2系統あります。教科書のベクトルは矢印付き、入試・大学レベルは太字が主流です。

| 用途 | LaTeX ソース | レンダリング | 用途別の使い分け |
| --- | --- | --- | --- |
| 矢印つき | `\vec{F}` | $\vec{F}$ | 高校教材の標準 |
| 太字（要 bm パッケージ） | `\bm{F}` | $\bm{F}$（KaTeX未対応の場合あり） | 大学・入試解答 |
| 単位ベクトル | `\hat{x}` | $\hat{x}$ | 座標方向 |
| 内積 | `\vec{A}\cdot\vec{B}` | $\vec{A}\cdot\vec{B}$ | スカラー積 |
| 外積 | `\vec{A}\times\vec{B}` | $\vec{A}\times\vec{B}$ | ベクトル積 |
| 大きさ | `\|\vec{F}\|` | $\|\vec{F}\|$ | ノルム |
| 成分表示 | `(F_x, F_y, F_z)` | $(F_x, F_y, F_z)$ | 直交成分 |
| ナブラ | `\nabla` | $\nabla$ | 勾配・発散・回転 |

<Callout variant="note" title="bm パッケージの代替">
KaTeX や Web では `\bm` が使えないことがあります。その場合は `\boldsymbol{F}` で代用：`\boldsymbol{F}` → $\boldsymbol{F}$。Overleaf 等の通常 LaTeX なら `\usepackage{bm}` で `\bm` が使えます。
</Callout>

## <span id="h2-calculus">LaTeX 偏微分 書き方｜微分・偏微分・全微分・積分</span>

**LaTeX 偏微分 書き方** は分子・分母をきちんと並べると、組版品質が一気に上がります。

| 用途 | LaTeX ソース | レンダリング |
| --- | --- | --- |
| 常微分（一階） | `\dfrac{dx}{dt}` | $\dfrac{dx}{dt}$ |
| 常微分（二階） | `\dfrac{d^2 x}{dt^2}` | $\dfrac{d^2 x}{dt^2}$ |
| 偏微分（一階） | `\dfrac{\partial f}{\partial x}` | $\dfrac{\partial f}{\partial x}$ |
| 偏微分（二階） | `\dfrac{\partial^2 f}{\partial x^2}` | $\dfrac{\partial^2 f}{\partial x^2}$ |
| 混合偏微分 | `\dfrac{\partial^2 f}{\partial x\,\partial y}` | $\dfrac{\partial^2 f}{\partial x\,\partial y}$ |
| 全微分 | `df = \dfrac{\partial f}{\partial x}dx + \dfrac{\partial f}{\partial y}dy` | $df = \tfrac{\partial f}{\partial x}dx + \tfrac{\partial f}{\partial y}dy$ |
| 不定積分 | `\int f(x)\,dx` | $\int f(x)\,dx$ |
| 定積分 | `\int_0^T v\,dt` | $\int_0^T v\,dt$ |
| 重積分 | `\iint_S \vec{E}\cdot d\vec{S}` | $\iint_S \vec{E}\cdot d\vec{S}$ |
| 線積分（閉曲線） | `\oint \vec{B}\cdot d\vec{\ell}` | $\oint \vec{B}\cdot d\vec{\ell}$ |
| 極限 | `\lim_{t\to 0}` | $\lim_{t\to 0}$ |

<Callout variant="tip" title="\\,（薄いスペース）の活用">
積分の `dx` の前には `\,`（薄いスペース）を入れると `\int f(x)\,dx` のように見やすくなります。物理の論文・教材ではほぼ必須の習慣です。
</Callout>
